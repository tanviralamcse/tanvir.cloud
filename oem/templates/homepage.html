Features of the OEM Prototype

Job Listings

OEMs can post job listings for technicians. Technicians can apply for jobs based on their skills and location.

Service Requests

OEMs can manage service requests submitted by customers and assign jobs to technicians for fulfillment.

Reviews & Ratings

Technicians and OEMs can leave feedback and rate each other, helping maintain quality and trust on the platform.

Technology Stack

The OEM Prototype is built using the latest technologies for an efficient and scalable platform.

Django

Our backend is built with Django, a powerful Python framework that allows rapid development and scalability.

HTMX

HTMX is used for creating dynamic, real-time interactions on the frontend, making the user experience seamless.

Tailwind CSS

Tailwind CSS is used to create a modern and responsive UI design with utility-first CSS classes.

AWS

The platform is hosted on AWS, providing reliable cloud services with excellent scalability options.
